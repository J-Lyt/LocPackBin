# Snowdrop LocPack Converter

A console application that converts **'.locpack'** files to **'.locpackbin'** binary

## Supported Games:
- Avatar: Frontiers of Pandora

## Supported LocPack Files:
- menus.locpack
- subtitles.locpack
- subtitles_nocc.locpack

## Instructions:
1. Modify ``.locpack`` file using **Notepad++** or any other text editor
2. Drag modified ``.locpack`` file onto ``LocPackBin.exe``
3. Place the converted ``.locpackbin`` file into ``blue\localization\packages\english\`` directory
