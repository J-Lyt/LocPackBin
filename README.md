# Snowdrop LocPack Converter

A console application that converts **'locpack'** files to and from binary format.

## Supported Games:
- Avatar: Frontiers of Pandora
- Star Wars: Outlaws

## Supported LocPack Files:
- menus.locpack
- menus.locpackbin
- subtitles.locpack
- subtitles.locpackbin
- subtitles_nocc.locpack
- subtitles_nocc.locpackbin

## Instructions:
### .locpack to .locpackbin
1. Modify ``.locpack`` file using a text editor such as **Notepad++**
2. Drag modified ``.locpack`` file onto ``LocPackBin.exe``
3. Place the converted ``.locpackbin`` file into ``<blue/helix>\localization\packages\<language>\`` directory
### .locpackbin to .locpack
1. Drag ``.locpackbin`` file onto ``LocPackBin.exe``
